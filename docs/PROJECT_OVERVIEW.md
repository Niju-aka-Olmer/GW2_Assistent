# GW2 Assistant — Project Overview

## Описание проекта

**GW2 Assistant** — это веб-приложение-ассистент для игры Guild Wars 2. Приложение позволяет игрокам просматривать и анализировать своих персонажей, билды, инвентарь и банк через удобный веб-интерфейс с интеграцией искусственного интеллекта (DeepSeek) для получения рекомендаций.

## Цели

- Предоставить игрокам удобный веб-интерфейс для просмотра игровых данных GW2
- Интегрировать AI-ассистента (DeepSeek) для анализа билдов и инвентаря
- Поддерживать русскоязычную локализацию для всех игровых предметов
- Обеспечить кэширование данных для быстрой работы
- Создать масштабируемую архитектуру для добавления новых функций

## Целевая аудитория

- Русскоязычные игроки Guild Wars 2
- Игроки, интересующиеся оптимизацией билдов и экипировки
- Игроки, желающие анализировать инвентарь для продажи на торговой площадке
- PvE и WvW игроки, желающие получать AI-рекомендации

## Ключевые функции

1. **Выбор персонажа** — просмотр всех персонажей учётной записи
2. **Просмотр билда** — специализации, навыки, экипировка с детальной информацией
3. **AI-анализ билда** — рекомендации от DeepSeek по улучшению билда
4. **Инвентарь и банк** — просмотр предметов с ценами и фильтрацией
5. **AI-анализ инвентаря** — рекомендации по продаже предметов

## Ожидаемые результаты

- Полностью функционирующее SPA-приложение
- Интеграция с GW2 API (v2)
- Интеграция с DeepSeek API
- Поддержка тёмной и светлой тем
- Русскоязычный интерфейс
- Производительность за счёт кэширования
- Деплой через Docker Compose
