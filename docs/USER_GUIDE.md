# GW2 Assistant — Руководство пользователя

## О проекте

**GW2 Assistant** — это веб-ассистент для Guild Wars 2. Он позволяет просматривать персонажей, экипировку, инвентарь, банк, а также получать AI-анализ билда, инвентаря и торговой площадки с помощью DeepSeek AI.

**Доступ:** http://192.168.1.180:3000

## 1. Авторизация

### Что это
При первом входе вы попадаете на страницу подключения API-ключа Guild Wars 2.

### Как использовать
1. Введите ваш **GW2 API Key** в поле ввода
2. Нажмите кнопку **Подключиться**
3. Если ключ валидный — вы попадёте на страницу выбора персонажа

### Требования к ключу
- Ключ должен иметь права: `account`, `characters`, `inventories`, `builds`, `unlocks`, `tradingpost`
- Если ключ невалидный или истёк — вы увидите сообщение об ошибке

### Как получить API ключ
1. Перейдите на https://account.arena.net/applications
2. Создайте новый ключ, включив все необходимые разрешения
3. Скопируйте ключ и вставьте его в приложение

## 2. Выбор персонажа

### Что это
После авторизации отображается список всех ваших персонажей с основной информацией: имя, раса, профессия, уровень, баланс монет.

### Как использовать
- **Выбор персонажа** — нажмите на карточку любого персонажа, чтобы перейти к его экипировке
- **Профессии** — у каждого персонажа есть иконка профессии (Страж, Воин, Некромант, и т.д.)
- **Валюта** — под именем персонажа отображается золото/серебро/медь
- **Кнопка "Выйти"** — в правом верхнем углу, чтобы отключиться и ввести другой API ключ
- **Кнопка смены темы** — ☀️/🌙 рядом с кнопкой выхода

### Навигация
В хедере (верхняя панель) доступны ссылки:
- **Персонажи** — вернуться к списку персонажей
- **Рекомендации** — AI-анализ билда и инвентаря
- **Аукцион** — торговая площадка

## 3. Экипировка (Build)

### Что это
Страница экипировки выбранного персонажа. Показывает все надетые предметы: броню, аксессуары, оружие, подводное снаряжение.

### Как использовать
- **Сетка экипировки** — все предметы разбиты по категориям: Броня, Аксессуары, Оружие (набор 1 и 2), Подводное
- **Категории**:
  - *Броня* — шлем, наплечники, куртка, перчатки, поножи, сапоги
  - *Аксессуары* — рюкзак, амулет, аксессуары, кольца
  - *Оружие* — два набора оружия (если есть)
  - *Подводное* — подводное оружие и маска
- **Наведение на предмет** — появляется всплывающая подсказка с:
  - Полным названием и редкостью (цвет)
  - Характеристиками предмета (Сила, Точность и т.д.)
  - Ценой покупки/продажи на аукционе
  - Чат-ссылкой (можно скопировать в игру)
- **Клик на предмет** — открывает модальное окно с полным описанием: все характеристики на русском, тип предмета, требования по уровню
- **Вкладки персонажей** — над сеткой экипировки можно переключаться между персонажами без возврата к списку

### Кнопка "Анализ билда"
- **Рядом с именем персонажа** есть кнопка **Анализ**
- DeepSeek AI анализирует ваш билд и выдаёт рекомендации:
  - Оценку текущего билда (Power DPS, Condi, Support и т.д.)
  - Рекомендации по улучшению экипировки
  - Советы по специализациям с иконками и ссылками на вики
  - Ссылки на готовые билды с metabattle.com
- Результат сохраняется в истории (до 3 последних анализов)

## 4. Инвентарь и Банк

### Что это
Просмотр инвентаря выбранного персонажа и общего банка аккаунта.

### Как использовать
- **Вкладки** — переключение между "Инвентарь" и "Банк"
- **Инвентарь** — показывает все сумки персонажа с предметами (вкладка разделена на сумки)
- **Банк** — содержимое общего банка аккаунта
- **Количество** — если предмет в стаке >1, отображается число в правом верхнем углу иконки

### Фильтры
- **Поиск по имени** — текстовое поле для фильтрации предметов
- **Фильтр по редкости** — выпадающий список: Мусор, Базовый, Тонкая, Отличная, Редкая, Экзотическая, Возвышенная, Легендарная
- **Фильтр по типу** — выпадающий список: Оружие, Броня, Аксессуар, Спина, Сумка, Расходник, Трофей, Материал, Улучшение, Гизмо и т.д.

### Взаимодействие с предметами
- **Наведение** — всплывающая подсказка с названием, редкостью, ценой аукциона
- **Клик** — модальное окно с полным описанием на русском
- **Цена** — под иконкой отображается цена продажи на аукционе (если есть данные)

### Кнопка "Анализ инвентаря"
- Анализирует содержимое инвентаря или банка через DeepSeek AI
- DeepSeek даёт рекомендации: что продать, что оставить, что использовать для крафта
- Результат сохраняется в истории

## 5. AI-рекомендации

### Что это
Общая страница для всех AI-анализов. Объединяет анализ билда и анализ инвентаря в одном интерфейсе.

### Как использовать
1. **Выберите персонажа** — из выпадающего списка
2. **Вкладка анализа** — "Анализ билда" или "Анализ инвентаря"
3. **Инвентарь или Банк** — если выбран анализ инвентаря, можно указать цель: Инвентарь или Банк
4. **Кнопка "Анализировать"** — запускает AI-анализ
5. **DeepSeek API ключ** — если ключ не настроен на сервере, появится поле для его ввода

### История анализов
- Все результаты сохраняются в истории (до 3 последних на тип)
- В боковой панели (слева) отображаются сохранённые анализы
- **Клик на сохранённый анализ** — просмотр результата без повторной генерации
- **Кнопка "Очистить историю"** — удаляет все сохранённые анализы
- **Крестик (×)** — удаляет конкретный анализ из истории

### Формат результата
Анализ отображается в форматированном виде:
- **Заголовки** — выделены золотым цветом
- **Иконки предметов/специализаций** — отображаются как изображения
- **Ссылки на вики/metabattle** — кликабельны
- **Советы и рекомендации** — структурированы по шагам

## 6. Торговая площадка (Аукцион)

### Что это
Поиск предметов на аукционе Guild Wars 2 с отображением цен и AI-анализом рыночной ситуации.

### Как использовать

#### Поиск предметов
1. Введите название предмета в поле поиска (минимум 2 символа)
2. Нажмите **Поиск** или Enter
3. Результаты отображаются в виде таблицы с иконками
4. **Пагинация** — кнопки "Назад" / "Вперёд" под результатами
5. **Важно**: искать нужно на **английском** языке (GW2 API возвращает английские названия)

#### Выбор предметов для отслеживания
1. В результатах поиска отметьте предметы чекбоксами
2. Нажмите **"Добавить выбранные"** — предметы попадут в список отслеживания (правая колонка)
3. Предметы в списке отслеживания автоматически обновляют цены каждые 30 секунд

#### Просмотр цен
- Для каждого отслеживаемого предмета показываются:
  - **Иконка**, имя, редкость (цветом), уровень, тип
  - **Цена покупки** (самая высокая цена покупки на аукционе)
  - **Цена продажи** (самая низкая цена продажи)
  - **Количество лотов** на покупку и продажу
  - **Спред** — разница между ценой покупки и продажи
- Цены отображаются в формате: **золото/серебро/медь**

#### Управление списком
- **Удалить** — кнопка × рядом с каждым предметом в списке
- **Очистить всё** — удаляет все предметы из отслеживания

#### AI-анализ аукциона
1. Добавьте предметы в список отслеживания (не менее 1 предмета)
2. Нажмите **"AI-анализ аукциона"**
3. DeepSeek AI проанализирует:
   - Текущую рыночную ситуацию по выбранным предметам
   - Советы по покупке/продаже
   - Анализ спреда и ликвидности
   - Возможность заработка (flipping)
4. Результат сохраняется в истории анализов

#### Обменный курс
- В правой колонке отображается текущий курс обмена **Gem → Золото**
- Автоматически обновляется каждые 60 секунд

## 7. Общие элементы интерфейса

### Профессии и расы
Все названия профессий и рас отображаются на русском языке:
- *Professions:* Страж, Воин, Инженер, Рейнджер, Вор, Элементалист, Мечница, Некромант, Ревенант
- *Races:* Асура, Чарр, Человек, Норн, Сильвари

### Редкость предметов
Цветовая маркировка:
| Редкость | Цвет |
|----------|------|
| Junk | Серый |
| Basic | Светло-серый |
| Fine | Голубой |
| Masterwork | Зелёный |
| Rare | Жёлтый |
| Exotic | Оранжевый |
| Ascended | Розовый |
| Legendary | Фиолетовый |

### Формат валюты
Золото/серебро/медь отображается в виде: `2з 34с 50м`

### Всплывающие подсказки (Tooltip)
При наведении на предмет отображаются:
- Название (цвет редкости)
- Характеристики (если есть)
- Цена на аукционе (покупка/продажа)
- Если это валюта — количество в формате золото/серебро/медь

### Модальное окно предмета
При клике на предмет открывается окно с полной информацией:
- Иконка, название, редкость, уровень
- Все характеристики на русском языке
- Тип предмета (Оружие, Броня, Аксессуар и т.д.)
- Цена продажи валютой
- Флаги предмета (Уникальный, Учётная запись и т.д.)
- Чат-ссылка для использования в игре

### Скелетоны (Skeleton)
При загрузке данных отображаются анимированные заглушки (skeleton loaders).

## 8. Настройки и управление

### Смена темы
- Нажмите ☀️/🌙 в правом верхнем углу хедера
- Доступны: тёмная тема (по умолчанию) и светлая тема

### DeepSeek API Ключ
- Если на сервере не настроен DeepSeek API ключ, приложение попросит ввести его
- Ключ сохраняется в браузере (localStorage/sessionStorage)
- Вы можете ввести свой ключ в модальном окне, которое появляется при первом AI-анализе
- **Где получить:** https://platform.deepseek.com/api_keys

### Очистка кэша
- Сервер автоматически кэширует данные предметов, цен и персонажей
- Кэш обновляется по TTL (Time To Live)
- При перезапуске сервера кэш предметов (имена для поиска на аукционе) загружается с диска

### Выход из аккаунта
- Нажмите **"Выйти"** в правом верхнем углу
- API ключ GW2 будет удалён из сессии
- Вы вернётесь на страницу ввода ключа

## Решение проблем

### "Ошибка загрузки персонажей — Missing Authorization header"
1. Нажмите **"Повторить"**
2. Если не помогает — нажмите **"Выйти"** и введите API ключ заново

### "Не загружаются иконки предметов"
- Проверьте интернет-соединение (иконки загружаются с серверов ArenaNet)
- Если иконка не загрузилась — отображается серый квадрат

### "Поиск на аукционе не находит"
- Проверьте, что кэш предметов загружен (подождите ~60 секунд после перезапуска сервера)
- Ищите на **английском** языке, например: "Sword", "Greatsword", "Axe", "Helm"
- После первой загрузки кэша поиск работает мгновенно

### "AI-анализ не работает"
- Убедитесь, что введён валидный DeepSeek API ключ
- Проверьте баланс на https://platform.deepseek.com
- Попробуйте повторить запрос через минуту

### "Страница не открывается"
- Проверьте, запущен ли сервер (обратитесь к администратору)
- Убедитесь, что порты 3000 (фронтенд) и 8000 (бекенд) открыты
